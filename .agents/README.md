# Agent Workspace

This directory is the canonical repo-local workspace for agent guidance.

- `../AGENTS.md` is the root source of truth for coding-agent instructions.
- `rules/repo.md` links the local rules surface back to the root guide.
- `commands/README.md` records repo-local command guidance when it needs more detail than the root guide.
- `skills/README.md` records whether this repo has local skills and where they live.

Keep this directory lightweight and factual. Do not store secrets, transcripts, build output, or tool-managed state here.
